Here is some ELI5 background first about what a CSS Selectooooooooor is: When you dress up a website in CSS, it makes sense to have different styling across the elements. Eg you may want to have the word 'Blog' crossed out bc it is repulsive and replaced with 'Journal'.(This is not a blog by the way, it's a DevAcademy assignment and will not see the light of day)

The two selectors you can use would be:

  • 'class' and 'id'
    • or, what I remind myself of as:
  • . and #

The Class selector lets you to create style rules that apply to any element with a class attribute equal to a certain value. It's represented by a full stop or what American's call "a period - ."

The ID selector by contrast is "unique", applying to just one element on the page. Uno solo, or rather #unosolo.

What are the differences between relative, absolute, and fixed positioning?

CSS Flow Layout was described as "like a congo line" for the elements on a web page. I dont think I can improve on that analogy, so why try.

Static positioning is kind of the default positioning for elements. As we know in game theory defaults are very important. You could call this static positioning:

  1. css
    .static {position: static;}
    2. or just "none"thing at all - which assumes the default (I assume).

  2. css
    .static {/*none!*/}

Fixed positioning is when elements are positioned according to the viewer and not the page layout. Thus the elements stay in the same place as you scroll down a page. Like one of those horrible Herbalife pop ups on the Dailymail that follows you down the page as you death scroll.

Fixed positions will disappear when out of the view port's vision, but fixed elements will still remain in place. This is also useful for elements that you don't want to disappear from view regardless of where the user has navigated. Examples of this could include navigation, scroll bars and le hamburger.

Relative positioning will move the element as it relates to another/other elements.

Absolute positioning means you have taken an element "out of the congo line" "out of the flow", which then changes the default and current of the other elements.
